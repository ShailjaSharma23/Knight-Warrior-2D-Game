# 2D Platformer Game

A pixel-art 2D platformer game developed in Unity using C#. This project focuses on building polished platforming mechanics, combat, enemy AI, UI systems, and level progression while following game development best practices.

## Features

* Smooth player movement
* Jumping
* Wall jumping
* Melee sword attack
* Fireball shooting system
* Enemy AI with patrol and attack behavior
* Traps and environmental hazards
* Health system
* Three-heart UI
* Death and respawn system
* Level checkpoints
* Pause menu
* Background music and sound effects
* Volume controls
* Smooth camera follow
* Pixel-perfect platformer mechanics
* Animations and visual effects

## Tech Stack

* Unity
* C#
* Visual Studio Code
* Git & GitHub

## Current Status

The core gameplay systems have been completed. The project is now focused on designing and adding new levels, improving game balance, and polishing the overall experience.

## Controls

* **A / D** → Move
* **Space** → Jump
* **Left Click** → Fireball Attack
* **E** → Sword Attack
* **Esc** → Pause / Resume Game
